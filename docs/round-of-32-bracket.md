# 2026 FIFA World Cup — 32강 대진표 규칙

> **랜덤이 아닙니다.** FIFA가 사전에 정한 고정 브라켓 규칙에 따라, 조별리그 결과가 나오면 대진이 자동으로 채워집니다.

---

## 1. 토너먼트 구조

```
48팀
 └─ 12개 조 (A ~ L), 조당 4팀
      ├─ 각 조 1·2위  → 24팀 (자동 32강 진출)
      └─ 각 조 3위 중 상위 8팀 → 8팀 (추가 진출)
           ─────────────────────────
                    32팀 → 32강 (16경기)
                         → 16강 (8경기)
                         → 8강 (4경기)
                         → 준결승 (2경기)
                         → 결승 + 3·4위전
```

| 단계 | 경기 수 | 진출 팀 수 |
|------|---------|-----------|
| 조별리그 | 72경기 | 32팀 |
| 32강 (Round of 32) | 16경기 | 16팀 |
| 16강 (Round of 16) | 8경기 | 8팀 |
| 8강 (Quarter-finals) | 4경기 | 4팀 |
| 준결승 (Semi-finals) | 2경기 | 2팀 |
| 결승 (Final) | 1경기 | 우승 1팀 |

---

## 2. 32강 진출 방식

### 2-1. 조 1·2위 (24팀) — 자동 진출

각 조 상위 2팀은 **별도 선발 없이** 32강에 진출합니다.

### 2-2. 조 3위 (8팀) — 성적 순 선발

12개 조의 3위팀 중 **상위 8팀**만 추가로 32강에 진출합니다.

**순위 결정 기준 (동일 순서):**

1. 승점 (pts)
2. 득실차 (goal difference)
3. 다득점 (goals scored)

4팀은 탈락합니다.

## 3. 32강 대진 — 고정 브라켓

32강 16경기의 대진은 **FIFA가 미리 정한 슬롯**입니다.  
각 경기에는 "어느 조의 몇 위"가 들어갈지가 고정되어 있습니다.

### 표기법

| 표기 | 의미 |
|------|------|
| `1A` | A조 1위 |
| `2B` | B조 2위 |
| `3rd` | 3위팀 슬롯 (어느 조 3위인지는 아래 규칙으로 결정) |
| `W73` | 73번 경기 승자 |
| `L101` | 101번 경기 패자 |

## 4. 3위팀 배정 — FIFA Annex C

3위팀 8팀이 **어느 경기의 3위 슬롯**에 들어가는지는 랜덤이 아니라,  
FIFA **Annex C** 규칙에 따라 **허용된 조 조합** 안에서 자동 배정됩니다.

각 3위 슬롯에는 "이 조 1위"가 기다리고, 상대 3위팀은 **특정 조들에서만** 올 수 있습니다.

| 슬롯 | 1위(고정) | 3위팀이 올 수 있는 조 |
|------|-----------|----------------------|
| M75 | E조 1위 | A, B, C, D, F |
| M78 | I조 1위 | C, D, F, G, H |
| M79 | A조 1위 | C, E, F, H, I |
| M80 | L조 1위 | E, H, I, J, K |
| M81 | G조 1위 | A, E, H, I, J |
| M82 | D조 1위 | B, E, F, I, J |
| M85 | B조 1위 | E, F, G, I, J |
| M88 | K조 1위 | D, E, I, J, L |

### 배정 알고리즘 (이 프로젝트 구현)

1. 12개 조 3위팀을 승점 → 득실차 → 다득점 순으로 정렬
2. 상위 8팀을 3위 진출팀으로 선정
3. **백트래킹(backtracking)**으로 8개 슬롯에 Annex C 규칙을 만족하며 배정
4. 유효한 배정이 없으면 해당 조합은 불가능

```javascript
// js/config/defaults.js
var thirdPlaceSlots = {
  M75: { source: '1E', allowed: ['A', 'B', 'C', 'D', 'F'] },
  M78: { source: '1I', allowed: ['C', 'D', 'F', 'G', 'H'] },
  M79: { source: '1A', allowed: ['C', 'E', 'F', 'H', 'I'] },
  M80: { source: '1L', allowed: ['E', 'H', 'I', 'J', 'K'] },
  M81: { source: '1G', allowed: ['A', 'E', 'H', 'I', 'J'] },
  M82: { source: '1D', allowed: ['B', 'E', 'F', 'I', 'J'] },
  M85: { source: '1B', allowed: ['E', 'F', 'G', 'I', 'J'] },
  M88: { source: '1K', allowed: ['D', 'E', 'I', 'J', 'L'] }
};
```

## 7. 한국(A조) 관점 예시
